# 🎓 GraduE8 - Kalkulator Punktów Ósmoklasisty

Prosta i intuicyjna aplikacja webowa pomagająca uczniom klas ósmych obliczyć punkty rekrutacyjne do szkół ponadpodstawowych.

## 🚀 Funkcje
- Obliczanie punktów za egzaminy ósmoklasisty.
- Przeliczanie ocen na punkty.
- Dodawanie punktów za świadectwo z wyróżnieniem, wolontariat i inne osiągnięcia.
- Responsywny design (działa na telefonach i komputerach).

## 🛠️ Technologie
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Node.js](https://nodejs.org/en)
